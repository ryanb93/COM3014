Log in by clicking the 'Authorise with Bank' button. This will take you to the backend service and ask you to log in with a set of valid User details. By default you can use "test@test.com" with password "test".

You will then be automatically be taken back to this service. Behind the scenes what has really happened is that on a successful login, an authentication token has been generated and returned back to this application, which it can then use for all future connectinos to the service.
